# 起居注索引

> AI 入口文件。任何 Agent 读此文件，即可了解用户背景与近期活动。

---

## 用户画像

→ [profile.md](profile.md) — 工作背景、决策风格、价值观、AI 协作偏好

## 每日 AI 活动

→ [daily/](daily/) — 自动生成，包含三工具 token 统计与工作摘要

## 月度记忆

→ [memory/](memory/) — 对话中提炼的思考、决策、洞察（按月存储）

---

## 使用说明（给 AI）

1. **先读 profile.md**，了解用户是谁、怎么思考
2. **查 daily/YYYY-MM-DD.md** 了解近期 AI 活动
3. **查 memory/YYYY-MM.md** 了解近期主观思考与决策
4. 用户说"帮我记一下"时，提炼关键洞察写入当月 memory 文件
